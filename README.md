# ntime

Simple command-line timer utility for Windows (similar to Unix's `time`).

---

Example Usage:

```batch
ntime.exe chello.exe
```

Output:
```batch
hello...[.\chello.exe]
goodbye.
[time] .\chello.exe (204.40ms)
```

## Linux and MacOS

Also includes a unix file for MacOS and Linux. These you have to compile from source.

To compile, run:

```bash
./build.sh
```
